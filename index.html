Primary Cost Analysis

We are currently spending $131,220/year on Google Workspace Enterprise Plus for 237 users while maintaining Microsoft 365 licenses at $44,820/year. This dual licensing creates unnecessary overhead. Migration to a consolidated M365 environment will eliminate $138,720 in annual costs while maintaining full Office functionality.

1

Licensing Optimization Strategy

Maintain existing M365 E5 licenses for 155 active users. Right-size licensing based on actual usage patterns while keeping full Office suite access across the organization. This approach leverages our current Microsoft investment.

2

High Storage User Management

Add Exchange Online Plan 2 ($8/user/month) for 5 users with storage >100GB: keri.kaiser (996GB), samantha.swayze (519GB), jacob.medley (208GB), preston.farrington (131GB), seiya.vogt (115GB). This provides adequate mailbox capacity.

3

User Account Cleanup Actions

Remove 36 users inactive since 2017-2021 to reduce licensing costs. Archive data for 2 inactive users with significant storage (290GB total). Review 31 users inactive 30-180 days with department managers for status confirmation.
